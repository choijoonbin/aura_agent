### 핵심 사용자 시나리오

**시나리오 1 :&#x20;****[**"AI의 상시 리스크 관제 (자율 탐지 및 이상 징후 분석)"**]**

* ID : [SC-001]

* 상황 : 수만 건의 전표가 인입되는 결산기, AI 에이전트가 배후에서 실시간으로 사내 규정 위반 패턴을 스캔할 때.

* 목표 : 담당자가 놓치기 쉬운 '심야/주말 변칙 지출'을 AI가 스스로 탐지하고, 법적/규정적 근거를 즉시 제시하여 감사 리스크를 사전 차단함.

* 사전 조건 : `규정집.txt` 기반 벡터 DB 구축 완료, 시나리오별 실시간 탐지 트리거 활성화.

* 상세 흐름

|    |                                   |                                                                                           |               |
| -- | --------------------------------- | ----------------------------------------------------------------------------------------- | ------------- |
| 단계 | 사용자 행동                            | 시스템 동작                                                                                    | 비고            |
| 1  | 진행자가 '규정 위반 시나리오' 데이터 인입 (시연 제어판) | 전표의 결제 시간, 업종, 금액을 추출하여 학습된 **사내 경비 지출 관리 규정**과 실시간 대조                                    | AI의 자율적 모니터링  |
| 2  | 통합 워크벤치 리스트에서 신규 케이스 확인           | 이상 패턴(심야 사용/비업무 업종)을 감지하여 고위험 케이스로 자동 분류 및 알림                                             | 실시간 리스크 포착    |
| 3  | 케이스 상세 페이지 진입 및 추론 결과 확인          | **Reasoning Chain:** "제11조 2항(업무시간 외 결제 금지) 위반 확인. 주점 업종 사용으로 업무 연관성 부재 판단. 전액 환수 조치 권고." | 근거 기반 의사결정 지원 |

* 입력 예시

2026-02-14(토) 23:45 / '동경 이자카야' / 220,000원 / 적요: "대외협력 미팅"

* 기대 출력

"본 건은 토요일 심야 시간대(23:45) 주점에서 발생한 건으로, 사내 규정 제11조 2항을 위반하였습니다. AI 분석 결과, 업무 연관성이 낮으므로 전액 환수 조치를 제안합니다."

* 성공 기준

  * 기준 1: 기준 1: AI가 단순 알람이 아닌, **정확한 규정 조항(제11조)**&#xC744; 인용하여 판단 근거를 제시하는가?

  * 기준 2: 마스터-디테일 구조를 통해 리스트와 상세 근거를 단일 뷰에서 제공하는가?

**시나리오 2 :****&#x20;**"AI의 자율 조치 가이드 (정책 준수 및 자기 교정)"

* ID : [SC-002]

* 상황 : 전표 처리 과정에서 인당 한도 미준수 등 단순 휴먼 에러가 발생하여 재무 건전성이 저해될 우려가 있을 때.

* 목표 : AI가 초과 집행된 금액을 자동 계산하고, 후속 조치(환수/소명)를 제안함으로써 재무팀의 판단 리드타임을 획기적으로 단축함.

* 사전 조건 : `규정집.txt` 내 '제14조(인당 식대 한도)'에 대한 임베딩 데이터 활성화.

* 상세 흐름

|    |                         |                                                                            |                      |
| -- | ----------------------- | -------------------------------------------------------------------------- | -------------------- |
| 단계 | 사용자 행동                  | 시스템 동작                                                                     | 비고                   |
| 1  | 진행자가 '한도 초과 위반 시나리오' 생성 | 전표 내 인원수와 총액을 파싱하여 인당 평균 단가를 사내 한도와 대조 분석                                  | 수치 정합성 조사            |
| 2  | 워크벤치 내 AI 추론 탭 확인       | 위반 액수를 산출하고, 규정에 명시된 **조치 절차(제12조)**&#xC5D0; 따라 '환수' 또는 '설명서 제출' 가이드 자동 생성 | rescriptive Guidance |
| 3  | AI 제안 기반 조치 확정          | AI가 작성한 초안을 검토 후, 클릭 한 번으로 담당자에게 소명 요청 (메일,알람 등은 구현 제외)                    | 인간-AI 협업(HITL)       |

* 입력 예시

인원 2명 / 결제액 70,000원 / 적요: "대외 파트너 식사"

* 기대 출력

"제14조 2항(인당 2만원 한도) 위반. 1인당 평균 35,000원이 집행되어 총 30,000원의 한도 초과가 발생했습니다. 규정에 따라 초과분에 대한 환수 또는 사유 소명을 권고합니다."

* 성공 기준

  * 기준 1:AI가 위반 액수를 정확히 산출하고 논리적인 후속 프로세스를 제안하는가?

  * 기준 2:사용자가 AI의 판단 근거를 신뢰하여 즉각적인 의사결정이 가능한 수준인가?
